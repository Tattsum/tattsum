# Macの `$ open` コマンドをUbuntuでも使いたい！！

あれ，便利ですよね！

Terminalの作業からFinderを開きたいときとか結構需要があります．

それなので，Ubuntuでは，

# `$ xdg-open`

のコマンドが使えます！


### ヤッター！！！

`$ apt install xdg-utils` とすれば，コマンドを使えるようになります！
